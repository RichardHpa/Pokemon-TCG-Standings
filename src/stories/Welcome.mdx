# Pokemon TCG Standings

This is a simple app to keep track of the standings of a Pokemon TCG Tournaments. It is built with React and uses the [Pokedata.ova](pokedata.ovh) to fetch data about the pokemon tournaments.

## Tech

The site uses [tailwindcss](https://tailwindcss.com/) for styling and [react-router](https://reactrouter.com/) for routing.
